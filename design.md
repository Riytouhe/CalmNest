# CalmNest Design System

## Brand
**App:** CalmNest — wellness, breathing, meditation, mood tracking
**Vibe:** Bold & modern. Dark, immersive, premium feel. Think high-end fitness app meets mindfulness.

## Colors
- Background: `#0A0A0F` (near-black)
- Surface: `#13131A` (card background)
- Surface2: `#1C1C28` (elevated cards)
- Accent Purple: `#7C5CFC` (primary CTA, active states)
- Accent Teal: `#00E5BF` (secondary, mood indicators)
- Accent Coral: `#FF6B6B` (warning, high energy)
- Text Primary: `#FFFFFF`
- Text Secondary: `#A0A0B8`
- Text Muted: `#50506A`
- Border: `#2A2A3A`

## Typography
- Font: Poppins (system fallback: sans-serif)
- Display: 32px / bold
- Title: 24px / semibold
- Heading: 18px / semibold
- Body: 14px / regular
- Caption: 12px / regular

## Spacing
- Base unit: 8px
- Screen padding: 20px
- Card padding: 16px
- Gap between sections: 24px

## Border Radius
- Card: 20px
- Button: 14px
- Pill: 100px
- Small: 10px

## Shadows/Glow
- Accent glow: shadowColor #7C5CFC, shadowRadius 20, opacity 0.4
- Teal glow: shadowColor #00E5BF, shadowRadius 16, opacity 0.3

## Iconography
Use phosphor-react-native icons, weight="duotone" for nav, weight="bold" for actions.

## Component Patterns
- Cards: dark surface, 20px radius, subtle border (#2A2A3A)
- Buttons: filled with accent gradient or solid accent color
- Progress rings: stroke-based circular SVG
- Charts: bar charts with teal/purple gradient bars
